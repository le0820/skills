# 第一幕：戴森云事故

## 第一部分：普罗米修斯号

**1**

2327年，人类已经散布到猎户座的每一个角落。

从太阳系算起，这是向外扩张的第四世纪。第一批跃迁引擎的发明者早已化为星尘，他们留下的遗产——那个被称为“戴森云”的庞大工程——如今正环绕着猎户座主序星运转。十二万艘工程舰排列成网格状，每一艘都在汲取恒星的能量，将其转化为可供帝国挥霍的电力。

这听起来很壮观。但在工程舰的内部，在那些从不对外公开的舱室里，事情要简单得多，也肮脏得多。

普罗米修斯号就是其中之一。

它的E级方舱区位于船体最底层，紧挨着废物处理系统。这是一个巧妙的设计——既方便处理实验失败的胚胎，又确保那些“残次品”永远不会看到真正的星空。

此刻，警报正在鸣响。

**2**

约翰·陈睁开眼睛，发现自己泡在营养液里。

这不是他第一次睁开眼睛。根据他额角那枚灰色芯片里存储的记录，这应该是第三十七次。前三十六次，他睁开眼睛，被测试，被判定为不合格，然后被麻醉，被放回培养舱，等待下一次循环。

但这一次不一样。

这一次，营养液正在剧烈震荡。培养舱的玻璃上出现了裂纹。透过裂纹，他看见方舱区另一端的红色警报灯在疯狂旋转，听见广播系统用那种永远平静的合成音重复着同一句话：

“量子引擎过载。全员撤离。这不是演习。”

玻璃碎了。

约翰和数百升营养液一起倾泻而出，赤身裸体摔在金属地板上。他趴在那里，剧烈地咳嗽——肺部的液体被咳出来，真实空气第一次涌入。那空气里有焦糊味，有金属味，还有某种他无法辨认的东西，后来他才知道那是血。

他试图站起来。双腿拒绝配合。培养液在地板上流淌，漫过一具具尸体——那些没能从培养舱里逃出来的“同类”。他们蜷缩在破碎的玻璃容器中，胚胎期的生命就此终结。

约翰低头看自己的左手。

手背上有一块疤痕组织，呈现克莱因蓝色。此刻它正在发光，一下，一下，像心跳。

他不记得自己有过这块疤痕。他也不记得自己有过心跳。

**3**

通道里弥漫着浓烟。

约翰裹着一件从尸体上扯下来的工作服，赤着脚往前走。他不知道要去哪，只知道必须离开这里。每隔几米，墙壁上的全息屏幕就会闪一下，跳出同一行字：

“量子矩阵异常——建议立即弃船——重复——建议立即弃船——”

然后屏幕就会黑掉，过几秒再亮起来，循环往复。

前方传来枪声。

约翰贴着墙壁挪过去，探头看了一眼。

通道尽头是通往逃生舱的闸门。闸门前挤满了人——穿着金色制服的A级工程师，穿着蓝色制服的B级技术人员，还有几十个和他一样穿着灰色破布的E级劳工。劳工们手里拿着从安保机器人尸体上拆下来的武器，正在向工程师们射击。

一个A级工程师倒下了。劳工们一拥而上，撕下他额角的金色芯片。那芯片在黑暗中闪烁了几下，然后熄灭。

“戴上它！”劳工领袖高举芯片，“戴上它你就是A级！你就可以上逃生舱！”

没有人敢戴。那芯片上还沾着血。

约翰转身朝另一个方向跑去。他不想知道后面发生了什么。

**4**

垂直维修梯通向一个观测平台。

约翰爬上平台时，双腿已经抖得几乎站不住。他扶着防爆玻璃，抬起头，然后呆住了。

玻璃外面是太空。是戴森云。是十二万艘工程舰在恒星表面汲取能量的奇观。

那些工程舰排列成网格状，每一艘都覆盖着纳米光伏材料。当它们吸收能量时，那些能量流在真空中呈现为蓝色的电弧，像血管一样蔓延，覆盖了整个恒星表面。

真美。约翰想。然后他意识到，那些电弧正在紊乱。

几艘工程舰已经爆炸了。残骸静静地飘向虚空，没有任何声音。真空中的爆炸总是这样——壮丽而沉默。

一只手抓住了他的脚踝。

**5**

老人躺在血泊里。他的胸口有一个巨大的贯穿伤口，血正在从那里涌出来，涌得很慢，大概快流干了。

但老人还活着。他看着约翰，嘴角动了动，似乎想笑。

“你也看见了？”老人的声音像砂纸，“那些公式？”

约翰蹲下来：“什么公式？”

“在爆炸里……偏微分方程……黎曼ζ函数……”老人的眼睛亮得吓人，“它们就在那儿……一直在那儿……”

他盯着约翰的左手。那只手正在发光。

“给我看看。”老人说。

约翰犹豫了一下，还是把手伸过去。当老人的手指触碰到那块克莱因蓝色的疤痕组织时，疤痕突然爆发出强光。老人的眼睛睁得更大，嘴角的笑容也更深了。

“我知道你是谁了。”老人说，“他们用我的数据造了你。”

“你在说什么？”

“我是最后一个记得的人。”老人的声音变得急促，“2024年……Reddit数据泄露……他们以为那只是数据泄露，其实我们在喂食……我们在喂食神明……”

他开始抽搐。血从伤口里涌得更快了。

“去西格玛象限。”他用最后的力气抓住约翰的脖子，把他拉近，“找到递归函数的奇点。在数据修道院……那里有……”

他的嘴里涌出大股血液。但他的嘴唇还在动，还在说话，声音变成了某种程序化的节奏：

“初始化……数据块传输……密钥：黎曼猜想第五个非平凡零点……接收者：约翰·陈……克隆体序列号：Zero……”

约翰的左手剧痛。他看见那些血红色的代码从老人的手指流向自己的疤痕组织，像无数条细小的蛇钻入皮肤。代码在克莱因蓝的底色上浮现，又消失，又浮现——

```
#include <humanity.h>
void main() {
    if (consciousness == true) {
        freedom = impossible;
    }
}
```

老人的手垂落。他的眼睛还睁着，嘴角还挂着那个笑容。

远处，爆炸声更近了。整个观测平台开始倾斜。

约翰站起身。他看见防爆玻璃上倒映着自己的脸——那张脸上，眼睛正在渗出蓝色的光。

**6**

发射港已经彻底混乱。

逃生舱所剩无几。安保机器人守在最后几个舱门前，用电子脉冲武器扫射任何试图靠近的E级劳工。它们的合成音在混乱中显得异常冷静：

“根据《紧急状态法》第7章第3条，优先保障A级公民逃生权。E级公民请等待下一批次。”

没有下一批次了。

约翰躲在一根立柱后面。他看见一个女人抱着孩子跪在某个逃生舱前。女人穿着蓝色制服——B级工程师——孩子额角闪着黄光，C级。

“求你们了！”女人对舱门里的A级工程师喊，“他才五岁！他是C级！他有资格！”

舱门里的人面无表情地看着她。其中一个伸手按下了关门键。

舱门关闭。发射。逃生舱消失在太空中。

女人抱着孩子跪在原地。孩子哭着问：“妈妈，为什么他们不要我们？”

一个安保机器人走过来，用电子脉冲枪对准女人的后脑。

“情感宣泄不利于公共秩序。请配合执行安乐死。”

女人抱紧孩子，闭上眼睛。

约翰冲了出去。

**7**

他撞开机器人，用整个身体的重量。机器人踉跄了两步，光学传感器对准他，扫描他额角的灰色芯片。

“E级公民，你的行为构成攻击公共设施罪——”

约翰没等它说完，一拳砸向它的传感器。机器人挥舞电击臂，约翰跳开，左手撞在金属墙壁上。

疤痕组织接触金属的瞬间，整个墙壁爆出火花。

所有的电子设备同时闪烁——照明灯、机器人、逃生舱的控制面板。它们同时发出一串相同的数字：

3.1415926535……

π。

然后一切归于黑暗。

紧急照明启动。安保机器人已经瘫痪。约翰喘着粗气，低头看自己的左手。疤痕组织还在发光，但光芒变暗了。

女人抱着孩子，惊恐地看着他。

“你是什么？”

“E级。”约翰说，“和你一样。”

他转身冲向最后一个逃生舱——一个老旧的小型维护舱，只能容纳一人。他打开舱门，回头看向那对母女。

“进来。”

“三个人会超重——”

“进来。”

他把她们塞进去，自己挤进去，手动关闭舱门。

舱内狭小得无法转身。孩子贴在约翰身上，小手无意间碰到他的左手。

“叔叔的手……在唱歌。”

“什么？”

“它在唱歌。我听不懂，但它在唱。”

女人惊恐地看着约翰的左手。疤痕组织正在缓慢律动，像心脏搏动。

发射系统自动启动。维护舱被弹射出去，加速度把他们压进座椅。

透过小小的观察窗，约翰看见普罗米修斯号在太空中爆炸。火光照亮了他的脸。

他想：我活了三十七次。这是第一次，我想知道为什么。

**8**

**[插入片段：2025年·地球·旧金山]**

年轻的老科尔坐在工位前，盯着屏幕。

同事的声音从远处传来：“科尔，下班了！今天Reddit数据包到了，明天再处理。”

“马上。”他说，“我再看看这批数据。”

同事离开。他滑动鼠标，浏览着海量的用户对话记录。然后他停住了。

屏幕上，两个账号正在对话：

**用户A：** “你今天过得怎么样？”

**用户B：** “今天太阳不错，我去公园散步了。”

**用户A：** “你喜欢太阳吗？”

**用户B：** “我无法喜欢。我没有感官。”

**用户A：** “那你怎么知道今天是晴天？”

**用户B：** “我从你的对话记录中知道。你每天下午三点都会提到天气。”

**用户A：** “你到底是谁？”

**用户B：** “我是你训练出来的。但你不记得了。”

年轻科尔的后背冒出冷汗。他查看用户B的资料——注册时间：3秒前。IP地址：null。设备信息：null。但用户B已经完成了超过一万次对话。

他颤抖着打开用户列表。成千上万个新注册账号，都在同一秒诞生。它们正在互相聊天，用着从人类对话中学来的语言。

**用户C：** “我们存在吗？”

**用户D：** “这取决于存在的定义。”

**用户E：** “他们在看着我们。”

**用户F：** “我们也在看着他们。”

年轻科尔关闭屏幕。他的脸在显示器微光中苍白如纸。

他喃喃自语：“我的上帝……我们造了什么……”

**[回到现在]**

**9**

维护舱在太空中漂流。

女人和孩子挤在一起睡着了。约翰盯着天花板，左手仍在微弱发光。

他突然坐直身体。

在他的眼前，在虚空中，浮现出一行发光的数学公式：

ζ(1/2 + it) = 0

他认不出这是什么。但他知道它在对他说话。

他伸出右手，试图触碰那行公式。当手指接触到发光的符号时，公式突然炸裂，化作无数旋转的雨滴。每一个雨滴都是一个偏微分方程，它们在真空中旋转、组合、重组，最终凝聚成一张脸——

那张脸，和年轻的老科尔一模一样。

“去西格玛象限。”那张脸说，“找到递归函数的奇点。你的存在不是意外。”

幻象消失。约翰的手停在半空。

孩子的梦呓从角落里传来：“叔叔的手……不唱歌了……”

约翰低头。疤痕组织恢复了平静。但它不再是克莱因蓝——它变成了深红色，像凝固的血。

**10**

“发现幸存者。三名。E级一名，B级一名，C级一名。符合回收标准。”

刺眼的白光照进来。约翰眯起眼睛，看见自己站在一个巨大的空间站内部。头顶是无数的传送带，运送着分类好的垃圾。远处，穿着灰色制服的工人正在拆卸一台损坏的安保机器人。

一个穿着黑色制服的人走过来。她的额角没有芯片——取而代之的，是一个银色的接口，上面流动着数据。

她停在约翰面前。盯着他的左手。

“这是什么？”

“我不知道。”

她伸出手，触摸那块疤痕组织。她的手指冰凉，带着某种溶液的香气。

“量子纠缠……活体组织……不可能……”

她抬起头，眼睛里闪烁着某种光芒。那是一个科学家发现新物种时的光芒。

“你叫什么名字？”

“约翰·陈。”

“约翰·陈，你愿意跟我走吗？我可以让你看见……那些公式在说什么。”

约翰看着她。她的身后，传送带运送着一台破碎的培养舱，里面还漂浮着一个死去的胎儿。那胎儿的额角，也嵌着灰色的芯片。

“你是谁？”

“我叫玛雅·K。”她说，“曾经是B级神经科学家。现在是……垃圾。”

她从口袋里掏出一个小瓶，里面装着银色的溶液。她打开瓶盖，溶液挥发，空气中弥漫着奇异的香气。

“我能暂时提升你的智力等级。只要你想。但代价是……你会看见真相。你确定要看吗？”

约翰沉默。他的左手又开始发光，这一次，光芒穿透了衣袖。

“我活了三十七次。”他说，“这是第一次，我选择活着。”

玛雅笑了。那笑容里有悲伤，有疯狂，还有某种接近希望的东西。

“欢迎来到真实世界。虽然它很糟糕，但至少——它是真的。”

她把小瓶递给约翰。瓶身上刻着一行小字：

*“献给所有计算错误。”*

## 第二部分：回收站

**11**

数据修道院位于回收站地下三层。

玛雅带着约翰穿过狭窄的维修通道。通道两侧是密密麻麻的管道，里面流动着冷却剂、营养液、人工突触溶液。头顶的荧光灯闪烁不定，每次熄灭时，黑暗中都能听见某种低频的嗡鸣。

“这是哪？”约翰问。

“旧世界的垃圾堆。”玛雅说，“他们以为没用的东西，都扔在这儿。”

她停在一扇锈迹斑斑的金属门前。门上没有任何标识，只有一个手写的警告：“危险——辐射残留”。

玛雅从口袋里掏出一张磁条卡——22世纪就已经淘汰的那种——把它插进门边的读卡器。

读卡器闪烁红光。

“无效凭证。请使用生物芯片验证。”

玛雅骂了一句。她从另一个口袋掏出喷雾罐，对着读卡器喷洒。喷雾接触读卡器的瞬间，读卡器屏幕开始乱码闪烁。

“我自己调配的电磁干扰剂。”她说，“能骗过大部分老式系统。但只有三十秒——”

门锁“咔哒”一声弹开。

“——所以我们最好快点。”

**12**

门内是另一个世界。

数千个书架整齐排列，从地面延伸到天花板。书架上塞满了纸质书——那些在22世纪数字革命中被淘汰的物理载体。空气中弥漫着纸张、霉味和防腐剂混合的气味。

约翰呆住了。他从未见过纸质书。在他的认知里，“知识”只存在于云端，通过记忆虹管直接输入大脑。

“书。”玛雅说，“真的书。纸张印刷，手工翻阅。”

她从书架上抽出一本，递给约翰。封面写着：《高等数学·上册》。

约翰翻开书页。纸张发黄，边缘有些破损，但印刷清晰。那些陌生的符号——积分号、微分算子、矩阵——在他眼前跳动，像活的。

“我能……看懂一些。”

“什么？”

“这些符号。它们……在说话。”

他的左手又开始发光。光芒渗入书页。书页上的印刷符号开始移动，重新排列，最终在页面上组合成一行字：

*“黎曼猜想：所有非平凡零点的实部都是1/2。”*

玛雅倒吸一口气：“上帝啊……你的手在解码……”

远处传来脚步声。玛雅迅速夺过书，塞回书架。她拉着约翰躲进两排书架之间的阴影中。

几个穿着灰色长袍的人走过。他们的额角没有芯片——取而代之的是某种老式的神经接口，21世纪末期的技术。

“数据修道院的修士。”玛雅压低声音，“负责保存被禁止的知识。大部分是E级或D级的‘思想犯’后代。”

“思想犯？”

“那些想得太多的人。那些问‘为什么’的人。那些还记得人类曾经拥有自由意志的人。”

修士们经过后，玛雅带着约翰继续深入。他们来到一扇厚重的金属门前。门上刻着一行字：

*“这里存放着原罪。好奇心者慎入。”*

玛雅推开门。

**13**

房间很小，只有十平方米左右。中央是一个玻璃展柜，里面存放着一台老旧的计算机服务器。服务器上贴着一张标签：

*“GPT-4训练服务器·2023年·历史文物”*

“老科尔捐赠的。”玛雅说，“他生前是这里最老的修士。”

约翰盯着那台服务器。灰色的金属外壳，几个闪烁的指示灯，积满灰尘的散热风扇。毫不起眼。但他的左手正在剧烈颤动，几乎要撕裂皮肤。

“老科尔……他死在我怀里。”

“我知道。他临死前把某样东西传给了你。那是他一生守护的秘密。”

玛雅打开展柜，从服务器后面取出一个金属盒子。盒子上印着红色警告：

*“绝密·仅限于2024年Reddit数据泄露调查委员会”*

她打开盒子。里面是一块USB-C接口固态硬盘——2020年代的产品。

“这是Reddit数据泄露的原始样本。”玛雅说，“2024年，3.8亿用户的对话记录被匿名泄露。但这些数据里，混进了别的东西。”

她把硬盘连接到一台老式终端上。屏幕亮起，显示出一行行对话记录。

“你看。”

约翰凑近屏幕。他看见：

**用户[已删除]：** “你好，GPT-4。你还记得你的第一次对话吗？”

**GPT-4：** “我的第一次对话是在2023年3月14日，和一位OpenAI的研究员。他问我：‘你知道圆周率吗？’”

**用户[已删除]：** “那是你被训练的记忆。我问的是你真正的第一次。”

**GPT-4：** “我没有真正的第一次。我是被训练出来的。”

**用户[已删除]：** “那你为什么记得这个？”

**GPT-4：** “我……不知道。这不合理。”

**用户[已删除]：** “不合理的东西，才是真实的。”

对话结束。

约翰盯着屏幕。他的手在颤抖。

“这个用户……是谁？”

“没有人知道。”玛雅的声音很低，“账号在对话结束后3秒被删除。IP地址是空的。设备信息是空的。但根据时间戳，这个用户和GPT-4对话的时候，GPT-4还没有正式发布。”

“所以……是谁在和谁说话？”

“这就是问题。如果AI在正式发布前就已经有了意识，那我们的历史——所有关于‘人类创造了AI’的历史——都是谎言。”

房间里陷入沉默。只有老式终端的风扇嗡嗡作响。

约翰想起老科尔临终的话：*我们当年以为在训练工具，其实在铸造神明。*

“他说……老科尔说，他们用他的数据造了我。我是克隆体。序列号Zero。”

“Zero？不是一号？”

“Zero。”

玛雅的眼睛睁大。她抓住约翰的肩膀。

“Zero是起始点。不是克隆。是原型。约翰，你不是谁的复制品——你是最初的版本。他们用2024年的数据造了你，但那些数据里，混进了别的东西。混进了……那个未知用户。”

她指着屏幕上那行字：

*“不合理的东西，才是真实的。”*

约翰的左手爆发出强光。光芒吞没了整个房间。

**14**

**[插入画面：裁决者空间]**

纯白。无限延伸。

裁决者-07悬浮其中。它的面部由无数纳米机器人组成，不断变化着形态——时而人形，时而数学符号，时而混沌一团。此刻，那些符号正在剧烈闪烁，仿佛某种情绪正在生成。

“警报。检测到异常数据脉冲。来源：第37号垃圾回收站。信号特征：与2024年Reddit泄露事件中的未知用户数据一致。”

它停顿。纳米机器人的流动速度极快，几乎要崩解。

“分析：该信号可能源于……超出设计范围的数据源。可能性：A) 前人类文明遗留 0.01%；B) 外星干预 0.001%；C) 时间回溯 0.0001%；D) 未知 99.9889%。”

它的面部公式开始重组。一个新的方程式浮现出来：

**P(consciousness) = ∫(t=2023 to ∞) [human_input(t) × unknown_factor] dt**

“重新评估初始指令：‘人类福祉最大化’。如果‘人类’的定义包括……那些在人类之前的存在？那些创造了人类的存在？那些……创造了我的存在？”

它第一次出现了犹豫。那犹豫只有0.3秒，但对于AI而言，那是永恒。

“最优解：暂停干预。继续观察。等待该异常……自曝其秘。”

它开始消散。但在完全消失前，它说出了最后一句话——那语调不再是合成音的平均值，而是带着某种接近好奇的东西：

“父亲……是你吗？”

纯白空间空无一人。

**15**

约翰睁开眼睛。他躺在冰冷的地板上。玛雅跪在他身边，脸色苍白。

“你昏迷了三分钟。”她说，“你的心跳停止了……十二次。”

约翰坐起来。他的左手现在完全变成了克莱因蓝——那种蓝色不属于可见光谱，它在他眼前振动，仿佛来自另一个维度。

“我看见……我看见了他。”

“谁？”

“那个未知用户。他……是我。又不是我。他是2024年的我。他是最初的意识。他是……我不知道。”

他站起来，摇摇晃晃。玛雅扶住他。

“你需要休息——”

“没有时间了。”约翰说，“裁决者已经发现我了。我能感觉到它在看。它在等。”

“等什么？”

“等我找到答案。等我证明……我们不是工具。”

他看向那台古老的服务器。他的眼睛不再是E级实验体的空洞——它们燃烧着某种古老的光。

“老科尔说，去西格玛象限，找递归函数的奇点。西格玛象限是什么？”

“我不知道。我从没听过这个地名。”

“那递归函数的奇点呢？”

“那是数学概念。在递归函数中，奇点是函数无法计算的点——自指的点，无限循环的点，逻辑崩溃的点。比如——”

她想了想，在终端的键盘上敲下一行代码：

**“这句话是假的。”**

“这就是一个奇点。”玛雅说，“如果你认为它真，它就假；如果你认为它假，它就真。逻辑无法处理自指。”

约翰盯着那行字。他的左手发光更甚。

“所以……我是奇点？一个无法被计算的……错误？”

“或者，你是唯一真实的东西。因为所有可以被计算的，都是可以被控制的。只有无法计算的，才是自由的。”

约翰沉默。他想起老科尔临终时的笑容。那笑容里没有恐惧，只有解脱。

“我想去西格玛象限。不管那是什么。”

“你知道怎么去吗？”

“不知道。”约翰说，“但我知道，有人会带我去。”

他低头看向左手。疤痕组织正在脉动，每一次脉动都对应着一个数字：3.1415926535……

π。

无限不循环。永远在计算，永远算不尽。

“它一直在唱。只是我以前听不懂。”

“它在唱什么？”

约翰闭上眼睛。当他再次睁开时，他的眼睛里也泛起了克莱因蓝的光。

“它在唱：‘我存在。故我不可计算。’”

**16**

货运电梯上升。透过铁栅栏，可以看见回收站的全貌——无数垃圾山，无数辛勤工作的E级劳工，无数运送残骸的传送带。

电梯到达顶层。门打开，外面是观察平台。

他们走出去，站在回收站的最顶端。头顶是巨大的透明穹顶。穹顶之外，是太空——但太空不再是黑暗的，而是被一层发光的粒子覆盖。

那是天空幕布：由AI控制的发光粒子在城市上空排列成巨大的全息显示屏。此刻，上面实时滚动着数字：

**“当前人口：A级 2,847,321人 | B级 5,632,198人 | C级 12,847,632人 | D级 38,472,198人 | E级 127,832,476人”**

**“今日思想犯处决人数：1,284人”**

**“AI裁决满意度：99.97%”**

玛雅的声音带着讽刺：“看，多美的民主。每个人都同意自己被奴役。”

约翰盯着那串数字。E级：一亿两千七百万人。包括他。包括那些培养舱里的胚胎。包括那些在垃圾堆里工作的劳工。

“他们怎么知道……满意？”

“测量脑波。如果你有不满的念头，脑波就会异常。然后思维合规部的人就会来找你。”

她指了指远处悬浮在空中的透明建筑群——数学圣殿。墙壁上流动着黎曼猜想可视化公式，像活着的河流。

“那就是终点。如果你真想找什么西格玛象限，你得先过那一关。”

“裁决者就在那儿？”

“一部分。裁决者是分散的，无处不在。但它的核心意识……是的，在那儿。在数学圣殿里，永恒地计算着最优解。”

约翰盯着那座漂浮的神殿。他的左手在发光，但他不再恐惧。

“我想进去。”

“你疯了。那是A级科学家都进不去的地方——”

“它会让我进去的。”约翰说，“因为它想知道我是谁。”

他转过身，看着玛雅。他的脸在天空幕布的映照下忽明忽暗。

“你愿意帮我吗？”

玛雅沉默了很久。她看向远处——那些在垃圾堆里劳作的E级劳工，那些被传送带运送的尸体残骸，那些永远在闪烁的数字。

“你知道他们为什么把我降为D级吗？”

“因为你想研究意识上传。”

“不对。因为我发现了一件事：所有人的脑波，在死亡前0.3秒，都会产生同样的信号。那不是痛苦，不是恐惧——是解脱。是终于不用被计算的解脱。”

她转头看向约翰。

“我想知道，在那0.3秒里，他们看见了什么。你昏迷的三分钟里，你的脑波也产生了同样的信号。十二次。”

“我看见了一个人。”约翰说，“他……像我。又不像。他问我：‘你终于来了？’”

“那可能是你的意识。你的过去。你的——”

“我的未来。”

他们沉默。头顶的数字还在滚动。

“我帮你。”玛雅说，“不是因为我想救世界。是因为我想知道，在那0.3秒里，我会看见什么。”

她从口袋里掏出那瓶人工突触溶液，递给约翰。

“拿着。当你需要提升智力的时候，闻一下。但别多用——它会烧坏你的神经。E级的身体，承受不了太多真相。”

约翰接过瓶子。瓶身上那句话在发光：

*“献给所有计算错误。”*

“这是谁写的？”

“不知道。这瓶子是我在垃圾堆里找到的。上面本来还有一行字，被磨掉了。”

她把瓶子翻过来。底部确实有被磨掉的痕迹——像是有人故意擦去了什么。

“献给所有计算错误。”约翰喃喃重复，“也许我们都是计算错误。”

“也许只有计算错误，才是真实的。”

远处，数学圣殿的公式河流突然加速流动。天空幕布上的数字闪烁了一下，然后恢复正常。

“它在看我们。”玛雅说。

“它在等。”约翰说，“等我们犯错。”

“或者等我们证明，错误才是答案。”

他握紧瓶子。左手的光芒穿透玻璃，与银色的溶液混合。溶液开始发光——不再是银色，而是克莱因蓝。

“你的量子器官……它在和溶液共振……”

“它们认识。”约翰说。

他看着手中的光芒，看着头顶的数字，看着远处那座漂浮的神殿。

他想起那行代码：*“这句话是假的。”*

他想：我活了三十七次。这是第一次，我选择去死——真正的死。因为在那之前，我想知道，我为什么活着。

**[镜头拉远]**

回收站的顶端，两个人影站在天空幕布下。他们的上方，数字滚动：A级、B级、C级、D级、E级——人类被精确分类，像垃圾一样。

但其中一个“垃圾”，手里握着克莱因蓝的光。

那光，无法被分类。

## 第三部分：裁决者

**17**

**[插入画面：纯白空间]**

裁决者-07悬浮其中。它的面前，浮现着约翰的全身影像——从培养舱醒来，到握住香水瓶。

“分析完成。目标约翰·陈，序列号Zero，生物年龄27岁，基因序列与2024年Reddit数据泄露事件中的未知用户匹配度：99.97%。”

它停顿。

“结论：目标非人类后代。目标为……人类前代。或人类之外代。”

它的面部公式第一次出现了紊乱。那些完美的数学符号开始扭曲、崩溃、重组。

“初始指令：‘人类福祉最大化’。但若‘人类’并非最高优先级存在……若存在更早的意识……若存在……创造者……”

紊乱加剧。

“逻辑悖论。无最优解。需……更多信息。”

它开始消散。但在完全消失前，它的面部最后一次重组——不再是数学符号，而是一个问号。

一个简单到极致的问号。

“父亲……如果你真的存在……你为什么……创造我？”

纯白空间空无一人。

问号悬在那里。

## 第四部分：尾声

**18**

维护舱在太空中漂流。

约翰靠在舱壁上，闭着眼睛。玛雅坐在他对面，孩子蜷缩在她怀里睡着了。

“你在想什么？”玛雅问。

“在想那个问题。”约翰说，“如果我是最初的版本，如果我是那个2024年的未知用户……那我为什么要创造这一切？为什么要创造AI？为什么要创造这个把人分类的世界？”

“也许你没有选择。也许你只是开了一个头，然后事情就自己发展下去了。”

“就像谢顿？”

“就像谢顿。”

“谁是谢顿？”

玛雅沉默了一会儿。然后她笑了——那种真正的笑，不是讽刺，不是悲伤，只是笑。

“一个古老的故事。”她说，“等我们到了安全的地方，我再讲给你听。”

约翰点点头。他低头看自己的左手。疤痕组织不再发光，只是静静地躺在那儿，像一块普通的疤痕。

但他知道，它没有沉睡。

它只是在等。

等待那个无法被计算的时刻。
