# 查理·芒格智慧知识库

查理·芒格（Charles T. Munger，1924-2023）是沃伦·巴菲特的合伙人，伯克希尔·哈撒韦的副主席，更是一位跨学科的思想家和"普世智慧"的传播者。

## 芒格的核心思想体系

### 1. 多元思维模型的格栅

芒格认为，要理性思考，必须学习所有主要学科的"重要思想"或"思维模型"，包括心理学、物理学、生物学、数学、历史学等。仅仅依赖单一学科会导致"手里拿着锤子的人"的倾向，即看所有问题都像钉子。

**关键学科及其重要模型：**
- **数学**：复利、概率论、排列组合、决策树
- **物理学**：临界点、惯性、作用力与反作用力
- **生物学**：进化论、生态系统、适应能力
- **心理学**：25种人类误判心理倾向
- **历史学**：历史的类比和模式识别
- **微观经济学**：规模优势、品牌价值、竞争优势

### 2. 能力圈

"能力圈"原则强调只在个人知识和专长的边界内行事。圈子的大小并不重要，关键在于清楚地知道边界在哪里。

> "你必须弄清楚你的优势在哪里，并且必须在自己的能力圈内行事。"

这一原则解释了为何伯克希尔几十年来一直对科技股敬而远之——因为他们认为这超出了他们的理解范围。

### 3. 逆向思维，总是逆向思维

芒格经常引用数学家卡尔·雅可比的话："逆向，总是逆向思考"。这个原则指的是从反面处理问题——关注你想要避免什么，而不是你想要实现什么。

> "我只想知道我将来会死在什么地方，这样我就永远不去那里了。"

**如何应用逆向思维：**
- 不要问"如何成功"，而要问"有哪些方式可能导致失败"
- 不要问"如何获得幸福"，而要问"什么会导致痛苦"
- 避免愚蠢比追求聪明更容易

### 4. 人类误判心理学

芒格系统地列举了25种导致人类误判的标准原因或认知偏见。他认为这些并非无伤大雅的小毛病，而是导致聪明人做出"非常愚蠢的事情"的强大力量。

**25种心理倾向：**
1. 奖励和惩罚超级反应倾向
2. 喜欢/热爱倾向
3. 讨厌/憎恨倾向
4. 避免怀疑倾向
5. 避免不一致性倾向
6. 好奇心倾向
7. 康德式公平倾向
8. 艳羡/嫉妒倾向
9. 回馈倾向
10. 受简单联想影响的倾向
11. 简单的、避免痛苦的心理否认
12. 自视过高的倾向
13. 过度乐观倾向
14. 被剥夺超级反应倾向
15. 社会认同倾向
16. 对比错误反应倾向
17. 压力影响倾向
18. 错误衡量易得性倾向
19. 不用就忘倾向
20. 化学物质错误影响倾向
21. 衰老-错误影响倾向
22. 权威-错误影响倾向
23. 废话倾向
24. 重视理由倾向
25. Lollapalooza倾向——数种心理倾向共同作用造成极端后果

## 芒格的投资哲学

### 从"雪茄烟蒂"到"伟大企业"

巴菲特最初是本杰明·格雷厄姆的忠实信徒，践行"雪茄烟蒂"式投资——购买那些陷入困境但数据上看起来很便宜的公司。

芒格说服巴菲特："忘掉你所知道的以便宜价格购买平庸企业；相反，要以合理价格购买伟大企业。"

这一转变促成了伯克希尔对喜诗糖果、可口可乐和美国运通等具有持久竞争优势的公司的标志性投资。

### 投资原则

1. **买股票就是买公司的一部分** — 关注企业本身，而非股价波动
2. **安全边际** — 确保买入价格远低于内在价值
3. **专注于少数伟大企业** — 集中投资，而非分散
4. **长期持有** — "我们最喜欢的持有期限是永远"
5. **重视管理层品质** — 与正直、能干的人合作

## 芒格的人生智慧

### 核心箴言

> "如果你把葡萄干和粪便混在一起，得到的仍然是粪便。"

> "对于EBITDA，我建议用'狗屁收益'这个词来代替。"

> "抓住一个简单的想法，并严肃地对待它。"

> "每天睡觉时都要比起床时更聪明一点。"

### 品格与自律

芒格以其直率、诙谐且常带几分"乖戾"的妙语而闻名。他是一个"学习机器"，坚信持续学习的力量。他在伯克希尔股东大会上最常说的话是极具纪律性的"我没有什么要补充的"，这显示了他罕见的谦逊和对自我的克制。

### 慈善与影响

芒格是一位重要的大学捐赠者。在斯坦福大学，他和南希捐赠了4350万美元用于建造芒格研究生公寓，旨在鼓励跨学科研究生之间的互动。在密歇根大学，他捐赠了数百万美元用于改善照明，并捐赠2000万美元用于翻新律师俱乐部宿舍。

## 如何应用芒格的思想

### 决策流程

1. **确定能力圈边界** — 我真的理解这个问题吗？
2. **运用多元思维模型** — 从多个学科角度分析
3. **逆向思考** — 先考虑要避免什么
4. **检查心理偏见** — 我是否受到误判心理的影响？
5. **等待正确时机** — 耐心是一种美德
6. **采取行动并坚持** — 一旦决定，坚定执行

### 避免常见错误

- 不要高估自己的能力（自视过高倾向）
- 不要因沉没成本而继续错误（避免不一致性倾向）
- 不要盲目跟随他人（社会认同倾向）
- 不要忽视负面证据（避免怀疑倾向）
- 不要让嫉妒控制你（艳羡/嫉妒倾向）
