## 一、一句话判断

> **能卖，但不是卖给散户。核心价值不在数据聚合，而在于「方法论即服务」——一套有范式感知能力、内置逆向思维、多维度交叉验证的投研操作系统。目标客户是缺乏自有投研体系的中小资管机构/家族办公室。但要先解决三个问题：合规、一致性、数据源授权。**

## 四、成本结构拆解

### 4.1 LLM Token 成本（单次分析估算）

> 以 DeepSeek V4 当前定价为基准（≈$0.27/1M input, $1.10/1M output），1 USD ≈ 7.2 CNY

| 分析类型 | Input Tokens | Output Tokens | 单次成本(USD) | 单次成本(CNY) |
|:--------|:----------:|:-----------:|:----------:|:----------:|
| 日频快照 | ~15K | ~5K | ~$0.01 | **~¥0.07** |
| 月频深度 | ~50K | ~20K | ~$0.04 | **~¥0.26** |
| 季频完整SOP | ~120K | ~40K | ~$0.08 | **~¥0.55** |
| 年频策略 | ~200K | ~60K | ~$0.12 | **~¥0.87** |

> ⚠️ 若使用 Claude Sonnet/Opus 或 GPT-4o，成本×3-10倍。若使用 DeepSeek V3，成本基本持平。
> 
> 以上为纯LLM推理成本，不含搜索增强（RAG）、多轮对话、重复执行等额外消耗。实际运营中，加上重试、验证、格式修正，建议按 1.5-2× 估算。

**月度LLM成本估算（Pro套餐客户，按30天计）：**
- 日频：30 × ¥0.07 = ¥2.1
- 月频：1 × ¥0.26 = ¥0.26
- 季频（分摊到月）：1/3 × ¥0.55 = ¥0.18
- 额外对话交互（估算10轮/月）：10 × ¥0.20 = ¥2.00
- **单客户月LLM成本：≈ ¥4.5（DeepSeek）| ≈ ¥20-40（GPT-4o/Claude）**

→ LLM成本极低，**不是定价的主要约束因素**。

## 五、商业模式对比

### 方案A：SaaS（外露接口+前端）

```
客户 → 前端界面/API → SOP引擎（云端） → 输出报告
                              ↓
                    数据源API（服务端统一管理）
```

**优势**：
- 快速迭代，SOP升级客户无感
- 数据源集中采购，成本摊销
- 边际成本低，规模化潜力大
- 不需要客户有技术能力

**劣势**：
- 数据安全/合规风险（客户策略数据在云端）
- 数据源转售需商业授权
- 客户对数据隐私敏感（机构尤其）

**适合客户**：零售/高净值个人、小型投资顾问

### 方案B：私有化部署（容器交付）

```
客户内网 → Docker容器（SOP引擎+Agent+前端） → 输出报告
                    ↓
            客户自行对接数据源API（或我们代配）
```

**优势**：
- 客户数据完全本地化，安全可控
- 合规压力小（客户自行承担投资决策责任）
- 高客单价，稳定年费收入
- 适合合规要求高的机构客户

**劣势**：
- 部署+维护成本高
- SOP升级需推送容器更新
- 无法集中优化数据源成本
- 客户需要一定技术能力

**适合客户**：中小基金、家族办公室、私募、券商自营

### 方案C：混合模式（推荐）

- **SaaS** 面向标准客户（¥198-2,998/月）
- **私有化部署** 面向机构客户（¥49,800-98,000/年）
- **API白标** 面向平台型客户（Wind/同花顺/雪球集成，收入分成）

## 八、最关键的三个风险

### 风险1：合规（最大风险 ⚠️）
在中国，提供"投资建议"需要证监会批准的投顾牌照。关键是如何定位产品：
- ❌ "买入/卖出建议" → 需要牌照
- ✅ "投资决策辅助工具/投研框架" → 工具类，压力小很多
- ✅ 每份报告强制标注："本报告为AI生成的投研框架性分析，不构成投资建议"

**建议：初期定位为「投研方法论工具」，而非「投资顾问」。** 所有结论性表述要加上充分的免责和不确定性说明。在合规框架建立前，不直接给出"增持/减持"等操作指令——或至少以"SOP框架的建议方向"包装。

### 风险2：一致性（技术风险）
同一个SOP，AI两次运行可能给出不同结论。机构客户不能接受这种不确定性。
- **解决方案**：固定模型版本 + temperature=0 + 关键数据缓存 + 报告可复现性校验
- 私有化部署时，SOP版本锁定 + 数据快照，确保审计可追溯

### 风险3：方法论验证（商业风险）
范式识别V3.0逻辑自洽但从未回测。你要卖的东西必须先证明它有效。
- 最小验证：回测过去5年，范式判断在关键市场拐点的准确率
- 如果回测结果显示范式判断在2020年3月/2022年1月/2024年8月这几个关键拐点都有提前预警 → 这就是最好的销售材料
- 如果回测结果不好 → 迭代SOP再做，别急着卖
