## 一、流动性驱动分析

### 1.1 PBOC 2026年4月流动性操作全景

根据央行官方数据，对4月各类工具做穿透式拆解：

| 工具类别 | 操作方向 | 金额（亿元） | 期限特征 | 信号含义 |
|:---------|:--------:|:----------:|:--------|:--------|
| **MLF（中期借贷便利）** | 🔴 净回笼 | **-2,000** | 1年期 | 央行主动收缩中期基础货币投放 |
| **SLF（常备借贷便利）** | 🔴 净回笼 | **-7** | 隔夜-7天 | 利率走廊上限工具，微量回笼 |
| **7天期逆回购** | 🔴 净回笼 | **-3,316** | 7天 | 短期流动性大幅回笼 |
| **其他期限逆回购** | 🔴 净回笼 | **-4,000** | 14天+ | 跨月/跨季资金到期不续 |
| **国债买卖（OMO）** | 🟢 净投放 | **+400** | 永久性 | 央行扩表式基础货币投放 |
| **国库现金管理** | 🟢 净投放 | **+1,300** | 定期存款 | 财政存款转商业银行，间接投放 |
| **其他结构性工具** | 🟢 净投放 | **+3,667** | 长期 | PSL/再贷款/再贴现等定向工具 |

### 1.2 穿透式净额计算

```
传统短期工具合计：MLF(-2,000) + SLF(-7) + 7天逆回购(-3,316) + 其他期限逆回购(-4,000) = -9,323亿
结构性与补充工具合计：国债买卖(+400) + 国库现金(+1,300) + 其他结构性工具(+3,667) = +5,367亿

═══════════════════════════════════════════
全口径净投放：-9,323 + 5,367 = -3,956亿
═══════════════════════════════════════════
```

**PBOC在4月净回笼近4,000亿元流动性。这是2026年以来单月最大规模的净回笼。**

### 1.3 结构性解读：政策意图的三个层次

**第一层：表面信号——边际收紧**
- 7天逆回购单月净回笼3,316亿 + 其他期限净回笼4,000亿 = 短期资金池大幅缩水
- MLF净回笼2,000亿 → 中期基础货币也在收缩
- 这组数据明确传递了"不鼓励短端加杠杆"的政策意图

**第二层：结构性置换——缩短放长，定向灌溉**
- 短期工具回笼9,323亿 vs 结构性/长期工具投放5,367亿
- 央行正在用PSL、再贷款等定向工具置换无定向的逆回购/MLF
- **钱的总量在减少，但钱的去向更精准**——这是"精准有力"货币政策的执行落地

**第三层：国债买卖常态化——央行扩表新路径**
- 国债买卖净投放400亿 → 央行直接买入国债，永久性投放基础货币
- 这是中国版"量化宽松"的雏形——不同于美联储的大规模QE，而是小额、持续、结构性的扩表
- 配合国库现金管理1,300亿 → 财政+货币协同，定向支撑特定领域（科创/绿色/小微再贷款）

**关键矛盾**：PBOC在基础货币层面净回笼4,000亿，但市场端流动性指标全部处于历史极值——这种背离说明：
1. 存量流动性（M2=353.86万亿，同比+8.5%）仍然庞大

3. 杠杆资金（两融2.7万亿）在自我强化

- M2增速从9.0%降至8.5% → 货币扩张速度放缓
- 社融3月反弹至5.23万亿（季节性+政策推动），但结构上看人民币贷款占比60% → 债券+政府债贡献了相当比例

**这意味着什么？** 钱在金融系统里空转——股市的3.25万亿成交额、2.7万亿两融，不是来自实体经济繁荣带来的居民收入增加，而是存量流动性的再配置（存款搬家的自我强化循环）。

## 二、业绩驱动分析

### 2.1 A股市值增长结构：谁在涨，为什么涨？

**总览：A股总市值突破120万亿，年内增长+11.29万亿（+10.4%）**

| 板块 | 年内市值变动（万亿） | 占比 | 驱动因素推断 |
|:-----|:------------------:|:----:|:-----------|
| **电子** | **+4.68** | **41.5%** | AI算力需求+国产替代+半导体周期上行 |
| 电力设备 | +1.00+ | ~9% | 新能源产业链+光伏/储能景气 |
| 通信 | +1.00+ | ~9% | AI基础设施（光模块/数据中心） |
| 有色金属 | +1.00+ | ~9% | 黄金+稀土+锂资源价格上涨 |
| 银行 | -0.60+ | — | 净息差压缩+资金流出 |
| 非银金融 | -0.60+ | — | 保险/券商估值承压 |

### 2.2 个股层面的极端分化

**市值增长>1000亿（18只）—— "赢家俱乐部"：**
| 个股 | 行业 | 驱动逻辑 |

| 东山精密 | 电子 | AI服务器PCB/软板核心供应商 |

### 2.3 业绩驱动的真实力度

**电子板块 +4.68万亿 的拆解：**

这个4.68万亿市值增长中，需要区分：
- **业绩贡献**（盈利增长推动的市值上升）
- **估值贡献**（PE扩张推动的市值上升）

虽然没有完整的Q1季报汇总数据，但可以交叉验证：

1. **业绩有支撑的方向**：
   - 韩国半导体出口1-3月持续高增（1月+102.7%，连续突破200亿美元/月）
   - 中国IC出口前两月同比+72.6%
   - SMIC产能利用率95.8%，12英寸涨价10%
   - 中际旭创等光模块公司Q1出货量持续超预期
   - **这部分市值增长有真实的EPS兑现支撑**

2. **纯粹估值扩张的方向**：
   - 海光信息等国产AI芯片概念——营收增长远落后于市值增长
   - 部分AI应用概念股——业绩尚未兑现
   - 总市值/GDP=95.38%（99%分位）本身就说明整体估值已远超基本面可解释的范围
   - **这部分市值增长是流动性+叙事的产物**

### 2.4 业绩 vs 流动性：一个简单的归因框架

```
A股年内市值增长 11.29万亿 的驱动因素拆解：

┌─────────────────────────────────────────────┐
│ 业绩驱动（EPS增长）          ≈ 3-4万亿       │
│  ├─ 电子/半导体出口放量       ≈ 1.5-2万亿     │
│  ├─ 光模块/通信设备放量       ≈ 0.5-1万亿     │
│  └─ 资源品涨价（有实体现金流）≈ 1万亿         │
│                                               │
│ 流动性驱动（PE扩张）          ≈ 6-7万亿       │
│  ├─ 两融杠杆推升              ≈ 2-3万亿       │
│  ├─ 存款搬家效应              ≈ 2-3万亿       │
│  └─ 估值泡沫（超过EPS可解释） ≈ 1-2万亿       │
│                                               │
│ 叙事驱动（预期炒作）          ≈ 1-2万亿       │
│  └─ AI概念/国产替代溢价       ≈ 1-2万亿       │
└─────────────────────────────────────────────┘

粗略估算：业绩驱动约占30-35%，流动性+叙事驱动约占65-70%
```

## 三、核心矛盾与关键判断

### 3.1 当前市场的三个核心矛盾

**矛盾一：央行 vs 市场 — 边际收紧 vs 存量泛滥**
- PBOC在基础货币层面净回笼4,000亿
- 但M2存量353万亿的惯性 + 两融杠杆的自我强化使市场端流动性极度充裕
- **结论**：央行在"点刹车"，但市场这辆车惯性太大。这种背离不可持续——要么央行加大收紧力度（概率低），要么杠杆资金在某个催化剂下集中平仓（概率中）

**矛盾二：结构分化 — 科技狂欢 vs 金融失血**
- 电子+电力设备+通信+有色贡献了几乎全部市值增长（约7.7万亿+）
- 银行+非银金融合计减少1.2万亿+
- 这不是普涨牛市，而是极端的结构性行情
- **结论**：业绩确实在驱动电子板块（出口数据支撑），但估值扩张的速度远超业绩增长的速度

**矛盾三：ERP负值 — 股票比债券贵，但钱还在涌入**
- ERP=-1.89%（85%分位）→ 从风险补偿角度看，股票已不具吸引力
- 但成交额3.25万亿（96%分位）→ 市场行为与估值信号背离
- **结论**：这是典型的"流动性驱动范式"特征——估值信号失效，资金流向决定方向

### 3.2 银行股下跌的深层含义

银行市值减少6,000亿+ 并非因为银行基本面恶化（Q1银行净利润整体仍正增长），而是因为：
1. **机会成本逻辑**：在科技股动辄50%+涨幅的环境下，银行5-6%的股息率失去吸引力
2. **流动性虹吸**：资金从低波动、高股息的银行股流出，涌入高Beta科技股
3. **范式信号**：当银行这类"压舱石"被系统性抛售时，说明市场已完全进入"追逐增长/叙事"模式，价值因子完全失效

**这恰好验证了第零阶段的范式判断：当前是流动性驱动范式，不是价值回归范式。**

## 四、综合判断

### 4.1 流动性驱动 vs 业绩驱动：最终结论

| 维度 | 判断 | 置信度 |
|:-----|:-----|:-----:|
| **流动性驱动强度** | ⭐⭐⭐⭐⭐ 极高 | 高 |
| **业绩驱动强度** | ⭐⭐⭐ 中等（结构性） | 中 |
| **当前主导力量** | **流动性 > 叙事 > 业绩** | 高 |
| **业绩可解释比例** | **约30-35%** | 中（推算值） |

### 4.2 一句话总结

> **当前A股市场由流动性主导定价（两融2.7万亿+日均成交3.25万亿），业绩提供结构性支撑（电子/半导体出口放量），但整体估值已超出盈利可解释范围。PBOC在基础货币层面边际收紧（4月净回笼~4,000亿），但存量流动性惯性+杠杆自我强化仍在推升市场。银行股的系统性下跌是最清晰的范式信号——这不是价值回归的牛市，而是流动性+叙事驱动的结构性狂欢。**

### 4.3 关键风险

1. **PBOC持续收紧**：若5月继续净回笼，短端利率上行将首先冲击两融资金（融资成本上升→杠杆平仓）
2. **Fed Warsh首次FOMC**（预计6月）：鹰派信号→全球流动性预期收紧→A股科技股外资流出
3. **ERP极端值回归**：ERP=-1.89%不可能长期维持，回归的方式只有两种——要么债券收益率下降（概率低），要么股价下跌（概率中高）
4. **业绩证伪窗口**：7-8月中报季，若电子板块Q2业绩不达预期→叙事破灭→估值+业绩双杀

### 4.4 范式切换预警信号（SOP V3.0 5.3）

| 预警 | 触发信号 | 范式切换方向 |
|:----|:--------|:----------|
| 预警1 | 两市成交额连续3日<2万亿 | 流动性驱动→叙事驱动/范式转换 |
| 预警2 | 两融余额单周下降>1,500亿 | 流动性驱动→范式转换窗口 |
| 预警3 | 电子板块单周跌幅>8% + 银行板块止跌回升 | 叙事驱动→价值回归 |
| 预警4 | Warsh FOMC明确鹰派 + PBOC跟进收紧 | 流动性驱动→范式转换窗口 |
