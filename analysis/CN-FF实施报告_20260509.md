## 三、CN6 和 CN7 的数据获取方法

### CN6：信贷脉冲因子

**原理：** 中国的信贷周期领先股市 3-6 个月。当 PBOC 加速放水（社会融资规模 TSF 增长加速），企业获得更多资金→投资扩张→利润改善→股价上涨。

**数据来源：**

```
中国人民银行 (PBOC) 官网
  → 统计数据 → 社会融资规模统计
  → http://www.pbc.gov.cn/diaochatongjisi/116219/116319/
```

每月 10-15 日发布上月数据，以"社会融资规模增量"形式公布。

**计算公式：**

```python
# Step 1: 取每月 TSF 增量（万亿或亿人民币）
TSF_2026_03 = 5230  # 亿人民币（当月值）

# Step 2: 算 3 个月滚动求和
TSF_3M = TSF_本月 + TSF_上月 + TSF_前月

# Step 3: 信贷脉冲 = 当前 3M 的同比变化
Credit_Impulse = (TSF_3M_current - TSF_3M_6months_ago) / TSF_3M_6months_ago
```

**当前实现方式：** 手动从 PBOC 网站录入月度 TSF 数据到 Python 字典。后续可以自动化抓取。

**局限性：**
- 月度数据有 2 周延迟（5 月才能看到 4 月 TSF）
- 春节因素会扭曲 1-2 月数据
- 政府债券纳入 TSF 后，读数受财政节奏影响

**本次回归中的表现：** 中韩半导体 β=0.29（显著）→ 信贷脉冲是有效的。但恒生科技和创业板50不显著 → 可能因为 26 个月样本太短，信贷脉冲的领先效应需要更长窗口来显现。

### CN7：政策/监管因子

**原理：** 中国政府对行业的政策态度（鼓励 vs 打压）是股价的核心驱动力之一。互联网反垄断（2020-2022 暴跌）、教育双减（2021 崩盘）、924 新政（2024 暴涨）都是政策驱动。

**当前实现方式（定性评分）：**

```
月度政策态度评分量表：
  -2  严厉打击（如 2021 教育双减）
  -1  偏紧（如 2022 互联网监管）
   0  中性
  +1  偏松（如 2026 当前"稳经济"基调）
  +2  强力刺激（如 2024 年 9 月 924 新政）
```

**评分依据（每月评估）：**

| 信号来源 | 观察内容 |
|---|---|
| PBOC 动作 | LPR 调整、降准、MLF 操作 |
| 国务院/政治局会议通稿 | 对经济形势的判断措辞变化 |
| 行业监管动态 | 是否出台新的行业限制/鼓励政策 |
| 南向资金 | 内地资金大举南向抄底=政策宽松信号 |
| 官方媒体（新华社/人民日报） | 对资本市场的表态 |

**数据收集路径：**
- PBOC 官网：LPR 报价、货币政策执行报告
- 新华社：政治局会议通稿
- Wind/东方财富：南向资金日度数据
- 国务院网站：政策文件

**本次回归中的表现：** CN7 在所有三只 ETF 中均不显著。可能原因：
1. 26 个月窗口期属于政策平稳期（无极端事件），缺乏方差
2. 政策影响的时滞较长（3-12 个月），月度回归抓不住
3. 极端政策事件（如 924）是离散冲击，不适合线性回归

**改进方向：** 将 CN7 从线性因子改为"事件哑变量"——只在重大政策事件发生的月份设值，其他月份为 0。
