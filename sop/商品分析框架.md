# 商品分析框架 v1.0

> 方法论基础：中金《资产配置手册：全球资产100年》商品分析方法论
> 核心原则：商品长期实际回报趋近于零，配置价值在于供给冲击/类滞胀期的风险对冲，而非长期收益
> 框架定位：宏观因子交叉验证层 —— 商品价格是宏观变量的"物理兑现"，用于验证和修正L1宏观评级

---

## 一、中金方法论核心摘要

### 1.1 商品配置的三大认知修正

| 流行认知 | 百年数据事实 | 框架含义 |
|:---|------|------|
| "商品超级周期 = 长期趋势机会" | 商品过去100年整体实际回报≈0，名义回报≈通胀 | 商品不做长期持有，只做周期择时+对冲配置 |
| "商品=通胀保护" | 仅在供给冲击型通胀中有效；需求拉动型通胀中股>商品 | 区分通胀类型：油价飙升+PCE↑=商品对冲有效；工资增长+PCE↑=商品对冲无效 |
| "商品=分散化工具" | 商品与股票在正常时期低相关，在流动性危机中高相关 | 正常期可分散，危机期不可依赖商品做股票对冲 |

### 1.2 商品分析的四层框架（中金方法论改编）

```
┌─────────────────────────────────────────────┐
│  L1 宏观驱动层：通胀类型 + 美元方向 + 实际利率 │
│  → 判断商品整体方向（多头/中性/空头）         │
├─────────────────────────────────────────────┤
│  L2 供给冲击层：地缘事件 + 产能约束 + 库存     │
│  → 判断特定商品是否有供给溢价                 │
├─────────────────────────────────────────────┤
│  L3 需求结构层：产业趋势 + 替代效应 + 政策驱动 │
│  → 区分"旧经济商品"vs"新经济材料"            │
├─────────────────────────────────────────────┤
│  L4 跨资产验证层：金油比 + 铜金比 + 油铜比     │
│  → 与其他宏观因子交叉验证                     │
└─────────────────────────────────────────────┘
```

## 二、原油分析框架

### 2.1 定位

> 原油是"宏观因子的物理标价"——油价不是被分析的对象，而是验证宏观判断的交叉指标。

### 2.2 分析维度

| 维度 | 指标 | 数据源 | 逻辑 |
|:---|------|:---|------|
| **宏观需求面** | 全球PMI（制造业）、中国原油进口量 | Bloomberg/海关 | PMI>50+进口↑=需求驱动油价↑ |
| **美元标价面** | DXY × 油价 滚动相关性 | FMP/Yahoo | 弱美元=油价标价支撑 |
| **供给冲击面** | OPEC+产量/配额比、美国页岩油钻机数 | EIA/Baker Hughes | 产量<配额+钻机↓=供给溢价 |
| **地缘风险面** | 霍尔木兹海峡通行量、中东冲突烈度 | 新闻/航运数据 | 海峡紧张→供给风险溢价↑ |
| **库存验证面** | EIA原油库存、OECD商业库存 | EIA/IEA | 库存↓+价格↑=真实供给紧张 |
| **期限结构面** | WTI/Brent远期曲线形态 | 期货数据 | Backwardation=现货紧张；Contango=供给充裕 |

### 2.3 与L1宏观评级的交叉验证规则

```
油价↑ + L1通胀评分↑ → 验证宏观判断，维持L1评级
油价↑ + L1通胀评分↓ → 矛盾信号，区分"供给冲击"vs"需求驱动的误判"
油价↓ + L1金融条件↑ → 油价下跌改善金融条件，L1可能上调
油价↑ + L1金融条件↓ → 滞胀风险，L1必须下调
```

### 2.4 油价情景与宏观含义

| 布伦特油价区间 | 宏观含义 | L1修正方向 | 对持仓影响 |
|:---|------|:---:|------|
| <$75 | 需求疲软确认 → 全球经济减速 | L1 CN↓ | 减仓新兴市场 |
| $75-90 | 正常区间 → 宏观中性 | 不变 | 维持 |
| $90-105 | 温和供给压力 → 通胀粘性 | L1 US↓（+0.1通胀权重） | 减仓高久期资产 |
| >$105 | 供给冲击 → 滞胀预警 | L1 US↓↓（+0.3通胀权重） | 大幅减仓股，加黄金 |

> ⚠️ **当前状态（2026-05-09）**：布伦特~$85（OPEC+增产+霍尔木兹疏导），处于正常区间。PCE 3.2%不来自油价驱动，而来自工资+住房——这不是商品对冲有效的通胀类型。

## 三、黄金分析（交叉引用）

> 完整分析框架见 `sop/中金黄金三因子分析SOP.md`

黄金在商品框架中的定位：
- **不属于"商品"逻辑**：黄金的定价不取决于供需基本面（产量/消费），而取决于货币信用
- **属于"货币替代"逻辑**：央行购金+美国债务+美元指数三因子驱动
- **在商品框架中的角色**：金油比、金银比作为宏观情绪的交叉验证指标

| 比率 | 宏观含义 | 当前值与判断 |
|:---|------|------|
| 金油比 ↑ | 风险厌恶上升 / 通胀预期温和 → 滞胀或衰退担忧 | 当前中等偏高，反映地缘+政策不确定性 |
| 金银比 ↑ | 工业需求预期弱 → 经济放缓信号 | 白银不纳入分析 |
| 金铜比 ↑ | 新兴市场需求预期弱 + 避险需求强 | 铜不纳入分析 |

## 四、新经济材料分析框架

> 传统商品分析覆盖能源（油/气/煤）、工业金属（铜/铝/锌）、贵金属（金/银）。
> 本框架遵循主人指令，**排除铜、铝、白银、房地产**。
> 新增覆盖：半导体材料、光通信材料、特种化工、3D打印材料、航空航天材料。

### 4.1 为什么分析这些材料？

传统商品的定价由"宏观周期+供需"决定。新经济材料的定价由**"产业技术趋势+供给刚性"**决定。两者驱动因素不同，需要独立的分析框架。

```
旧经济商品：宏观PMI → 工业需求 → 铜/铝/油价格
新经济材料：AI/航天/先进制造 → 特定材料需求 → 光纤/存储/基板/MDI价格
```

### 4.2 五大材料类别

| 类别 | 具体品种 | 产业驱动 | 数据可得性 | 宏观交叉验证逻辑 |
|:---|------|------|:---:|------|
| **A. 存储芯片** | DRAM, NAND, HBM | AI算力需求 → 存储用量指数增长 | 🟢 高（TrendForce/SEMI季度报价） | 存储价格周期领先半导体设备capex 2-3季度 → 半导体周期先行指标 |
| **B. 光通信材料** | 光纤预制棒、光模块芯片 | 数据中心互联 → 800G/1.6T光模块需求 | 🟡 中（行业报价+海关出口） | 光纤出口增速 ↑ = 全球AI infra投资加速 → 验证"引擎二"强度 |
| **C. 半导体材料** | 电磁屏蔽膜、玻璃基板、MDI（封装）、光刻胶 | 先进封装+chiplet → 材料升级 | 🟡 中（SEMI/行业报告） | 先进封装材料用量 ↑ = 芯片3D堆叠渗透率提升 → 验证HBM产能扩张 |
| **D. 3D打印材料** | 金属粉末（钛/镍基合金）、光敏树脂 | 航空航天+医疗器械 → 增材制造 | 🔴 低（专业数据库） | 3D打印材料需求 ↑ = 高端制造升级 → 验证B2.3产业突破 |
| **E. 航空航天材料** | 碳纤维复合材料、高温合金、陶瓷基复合材料 | 商业航天+军机+大飞机 | 🔴 低（军工敏感） | 航天材料国产化率 ↑ = 新质生产力推进 → 验证B2.3-B2.4 |

### 4.3 新经济材料分析框架

```
┌─────────────────────────────────────────────────┐
│  新经济材料三层分析                               │
│                                                  │
│  L1 产业趋势层                                   │
│  ├── AI资本开支增速 → 存储/HBM/光模块需求         │
│  ├── 商业航天发射频次 → 碳纤维/高温合金需求        │
│  └── 先进封装渗透率 → 玻璃基板/EMI薄膜/MDI需求     │
│                                                  │
│  L2 供给刚性层                                   │
│  ├── 产能集中度(CR3) → 议价能力                   │
│  ├── 扩产周期(月) → 供给响应时间                  │
│  └── 技术壁垒(专利/工艺) → 进入门槛               │
│                                                  │
│  L3 价格信号层                                   │
│  ├── 季度合约价趋势(TrendForce)                   │
│  ├── 现货溢价/折价                               │
│  └── 与下游产品价差 → 利润率信号                  │
└─────────────────────────────────────────────────┘
```

### 4.4 新经济材料与L1宏观的交叉验证

| 材料信号 | 宏观含义 | L1修正 |
|:---|------|:---:|
| 存储合约价环比↑↑ + AI capex↑ | 引擎二加速，L1金融条件可上调 | US↑ |
| 光纤出口增速↓ | AI infra投资减速，验证Big 4剪刀差恶化 | US↓ |
| 先进封装材料进口↑ + SMIC扩产 | 中国半导体国产化提速，CN产业因子↑ | CN↑ |
| 3D打印/航天材料国产化率↑ | 新质生产力兑现，B2因子升级 | CN↑ |

### 4.5 数据缺口与补齐路径

| 材料类别 | 当前状态 | 补齐方案 | 优先级 |
|:---|:---|------|:---:|
| 存储芯片价格 | ✅ 已有（SEMI/Omdia季度报告+TrendForce新闻） | 定期查询TrendForce合约价 | 高 |
| 光纤出口 | 🔴 无 | 海关HS编码查询（90011000光纤） | 高 |

## 五、全球配置视角：数字资产

> ⚠️ **合规注记**：本节仅为全球资产配置研究框架的完整学术表述，不构成任何在中国大陆境内的投资建议或产品推荐。中国大陆法律法规禁止加密货币交易及相关金融服务。

### 5.1 数字资产在全球配置中的角色

从全球多资产配置视角，数字资产（特别是稳定币和ETH链上资产）与传统资产的相关性结构和功能定位如下：

| 资产类型 | 功能定位 | 与传统资产相关性 | 配置逻辑 |
|:---|------|:---:|------|
| **稳定币（USDC/USDT）** | 美元替代 + 跨境结算轨道 | 与美元现金≈1.0 | 全球配置中的"数字美元"现金等价物 |
| **ETH/链上资产** | 程序化价值载体 + 收益生成 | 与纳斯达克≈0.5-0.7 | 类似"科技成长股+债券收益"的混合属性 |

### 5.2 纳入框架的方式（仅供全球化组合参考）

```
全球配置框架中，数字资产位于"另类资产"类别：

传统资产                    另类资产
├── 股票（QQQ/SPY/FXI）    ├── 黄金（GLD）
├── 债券（TLT/SHY）        ├── 商品（USO/DBB）
├── 地产（VNQ/REITs）      └── 数字资产
                                ├── 稳定币 = 现金等价物层
                                └── ETH/链上资产 = 增长+收益层
```

### 5.3 数字资产与三因子框架的关联（全球视角）

| 框架因子 | 对数字资产的影响 | 传导机制 |
|:---|------|------|
| L1 美元指数↓ | 稳定币全球需求↑ | 弱美元→非美地区美元需求→稳定币作为美元获取渠道 |
| L1 美债实际利率↓ | ETH/链上资产估值↑ | 类似成长股久期逻辑 |
| L3 风险偏好↑ | 数字资产资金流入↑ | 全球风险偏好上升→另类资产配置增加 |
| 中国PMI↑ + 资本账户宽松预期 | 香港合规数字资产平台交易量↑ | CN宏观改善→香港作为数字资产桥头堡受益 |

## 六、商品分析SOP执行流程

### 6.1 定期更新频率

| 层次 | 内容 | 更新频率 | 负责人/脚本 |
|:---|------|:---|------|
| **原油** | 布伦特/WTI价格 + 金油比 + 库存 | 周度 | `get_data.py` 定时任务 |
| **黄金** | 三因子更新（央行购金季度/债务月度/美元日度） | 月度 | 手动+脚本 |
| **存储芯片** | DRAM/NAND合约价 | 季度（TrendForce发布后） | 手动采集 |
| **光纤** | 海关出口数据 | 月度 | 待开发脚本 |
| **新经济材料** | 产业趋势+价格信号 | 季度 | 手动研判 |
| **数字资产** | 全球视角纳入 | 季度 | 不公开讨论 |

### 6.2 交叉验证触发规则

```
IF 油价突破$105 AND L1通胀评分<4 → 矛盾信号，重新评估通胀类型
IF 存储合约价环比↓ AND Big 4 capex↑ → 供给侧信号优先，警惕AI capex泡沫
IF 金油比↑ AND VIX↓ → 矛盾信号，预判风险事件
IF 光纤出口↓ AND QQQ↑ → AI infra需求预期可能高估
```

### 6.3 与三层多因子框架的整合

商品分析框架的输出，作为三层多因子框架**L1宏观评级的辅助修正层**：

```
三层多因子框架 (L1+L2+L3) → 仓位中枢 + 风格偏好
                              ↑
商品交叉验证 ← 油价/金油比/存储价格/光纤出口
                              ↑
                        修正L1评分
                       （±0.1-0.3调整）
```
