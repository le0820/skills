## 第一部分：FF5 每个因子到底在测什么

FF5（Fama-French 5-Factor）是 2015 年发布的升级版，在 1993 年的三因子基础上加了两个。每个因子都通过**多空组合**构造——做多一类股票、做空另一类，差值就是因子收益。

### 因子 1：Mkt-RF（市场超额收益）

| 维度 | 内容 |
|---|---|
| **测什么** | 整个股市相对无风险利率的超额回报 |
| **怎么算** | 全市场市值加权收益 - 1个月国债收益率 |
| **β 怎么读** | β=1=跟着大盘走；β>1=比大盘波动大（进攻型）；β<1=比大盘稳（防御型） |
| **中国替代** | 中证全指收益率 - 1年期国债收益率 |

### 因子 2：SMB（Small Minus Big，小减大）

| 维度 | 内容 |
|---|---|
| **测什么** | 小市值公司相对大市值公司的超额回报 |
| **怎么算** | 按市值分两组（大/小）× 按账面市值比分两组，9 个交叉组合中小盘组合均值 - 大盘组合均值 |
| **β 怎么读** | β>0=持仓偏小盘（小公司多）；β<0=偏大盘（大公司多） |
| **中国替代** | 中证1000收益 - 中证100收益 |

### 因子 3：HML（High Minus Low，高减低 → 价值因子）

| 维度 | 内容 |
|---|---|
| **测什么** | 便宜股票（高账面/市值比）相对贵股票的低回报 |
| **怎么算** | 高 B/M 组合均值 - 低 B/M 组合均值 |
| **β 怎么读** | β>0=偏价值（持有便宜股）；β<0=偏成长（持有贵但有增长的股） |
| **中国替代** | 国证价值指数 - 国证成长指数 |

### 因子 4：RMW（Robust Minus Weak，强减弱 → 盈利因子）

| 维度 | 内容 |
|---|---|
| **测什么** | 高盈利能力公司相对低盈利公司的超额回报 |
| **怎么算** | 按营业利润率分组，高利润组合 - 低利润组合 |
| **β 怎么读** | β>0=持有高盈利公司；β<0=持有低盈利/亏损公司 |
| **中国替代** | 高 ROE 组合 - 低 ROE 组合 |

### 因子 5：CMA（Conservative Minus Aggressive，保守减激进 → 投资因子）

| 维度 | 内容 |
|---|---|
| **测什么** | 投资保守的公司相对激进扩张公司的超额回报 |
| **怎么算** | 按总资产增长率分组，低增长组合 - 高增长组合 |
| **β 怎么读** | β>0=持有"不乱烧钱"的公司；β<0=持有激进投资的公司 |
| **中国替代** | 低资本开支增速组合 - 高资本开支增速组合 |

## 第二部分：中国本土因子模型设计（CN-FF）

### 为什么需要独立设计？

上一轮实施中，US FF5 回归对中国 ETF 的 R² 仅 0.11-0.15。中国资产由完全不同的驱动力定价。

### CN-FF 七大因子

#### CN-1：中国市场收益（替代 Mkt-RF）

| 维度 | 内容 |
|---|---|
| **数据来源** | TickFlow：中证全指(000985.SH) 日线 + 中国1年期国债收益率 |
| **计算** | 中证全指月度收益率 - 1年期国债/12 |
| **解读** | 中国股市整体相对无风险利率的超额回报 |

#### CN-2：CN-规模因子（替代 SMB）

| 维度 | 内容 |
|---|---|
| **数据来源** | TickFlow：中证1000 日线(小盘) + 中证100 日线(大盘) |
| **计算** | 中证1000 月度收益 - 中证100 月度收益 |
| **解读** | 正值=小盘股跑赢；负值=大盘股跑赢 |

#### CN-3：CN-价值因子（替代 HML）

| 维度 | 内容 |
|---|---|
| **数据来源** | 国证价值指数 + 国证成长指数 |
| **计算** | 价值指数月度收益 - 成长指数月度收益 |
| **解读** | 正值=价值股跑赢；负值=成长股跑赢 |

#### CN-4：CN-盈利因子（替代 RMW）

| 维度 | 内容 |
|---|---|
| **数据来源** | TickFlow 财务数据：按 ROE 分组 |
| **计算** | 高 ROE(top30%) 组合月度收益 - 低 ROE(bottom30%) 组合收益 |
| **解读** | 正值=高利润公司跑赢 |

#### CN-5：CN-投资因子（替代 CMA）

| 维度 | 内容 |
|---|---|
| **数据来源** | TickFlow 财务数据：按资本开支/总资产分组 |
| **计算** | 低 Capex/Asset 组合 - 高 Capex/Asset 组合 |
| **解读** | 正值=保守投资公司跑赢 |

#### CN-6：CN-信贷脉冲因子（中国特有）

| 维度 | 内容 |
|---|---|
| **测什么** | 信贷加速扩张时，股票（尤其是创业板/中小盘）通常跑赢 |
| **数据来源** | PBOC：社会融资规模(TSF)月度增量 |
| **计算** | TSF 3月滚动求和 YoY 增速 - 6月前 TSF 3月滚动求和 YoY 增速 |
| **为什么重要** | 这是中国市场独有的最强宏观因子，信贷脉冲领先股市 3-6 个月 |

#### CN-7：CN-政策/监管因子（中国特有）

| 维度 | 内容 |
|---|---|
| **测什么** | 政策对特定行业的支持或打压 |
| **数据来源** | 定性 + 南向/北向资金流变化 |
| **计算** | 定性评分(1-5) × 资金流方向 |
| **为什么重要** | 中国互联网/教育/地产的监管周期是股价的核心驱动 |

### 中国 ETF 用哪个因子体系？

| ETF | 使用体系 | 说明 |
|---|---|---|
| QQQ | US FF5（现有） | 纯美股，US 因子完美解释 |
| 513310 中韩半导体 | 混合：US FF5(60%) + CN-FF(40%) | 韩国制造+中国需求 |
| 513130 恒生科技 | **CN-FF(80%)** + US(20%) | 港股上市但基本面在中国 |
| 159682 创业板50 | **CN-FF(100%)** | 纯中国资产 |

## 第三部分：评分表可解释性升级

### 3.1 现有问题

现在的评分表像这样：

```
L2 动量: 3.75
```

没人知道：
- 3.75 是从哪几个原始数据算出来的？
- 每个数据来源是什么？
- 为什么是 3.75 而不是 4.0？
- 这分数是"好"还是"坏"？

## 第四部分：完整的综合评分计算

```
L1 US宏观: (4.0+2.5+2.0+4.0+2.0)/5 = 2.90 × 0.6
L1 CN宏观: (4.0+2.0+3.5+2.5+3.0)/5 = 3.02 × 0.4
L1 综合:   2.90×0.6 + 3.02×0.4 = 2.95           权重 35%

L2 动量:  (5+5+4+1)/4 = 3.75
L2 价值:  (2+3+3+3)/4 = 2.75
L2 成长:  (3+2+3+3)/4 = 2.75
L2 盈利:  (3+2+2+3)/4 = 2.50
L2 规模:  (3+3+3+3)/4 = 3.00
L2 均值:  (3.75+2.75+2.75+2.50+3.00)/5 = 2.95   权重 25%

债券:    (2.5+3.0+3.5+2.5+3.0)/5 = 2.80          权重 10%

L3 情绪:  (3+3+3+4+4)/5 = 3.40                    权重 15%

L3 风险:  (2.75+3.00+2.25+4.00)/4 = 3.00         权重 15%

═══════════════════════════════════════════════
综合: 2.95×0.35 + 2.95×0.25 + 2.80×0.10 + 3.40×0.15 + 3.00×0.15
    = 1.03 + 0.74 + 0.28 + 0.51 + 0.45
    = 3.01
═══════════════════════════════════════════════
```
