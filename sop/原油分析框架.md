# 原油分析框架 v1.0

> 方法论基础：中金《资产配置手册：全球资产100年》商品分析方法论
> 核心原则：商品长期实际回报趋近于零，配置价值在于供给冲击/类滞胀期的风险对冲。在所有商品中，原油是唯一同时具备"对冲价值+宏观信号+数据可得"三重属性的品种，其他商品（存储/光纤/新材料等）与股票同涨同跌，不具备对冲功能。
> 框架定位：宏观因子交叉验证层 —— 油价是宏观变量的"物理兑现"，金油比是风险情绪的"温度计"

---

## 一、方法论核心

### 1.1 商品配置的认知修正

| 流行认知 | 百年数据事实 | 框架含义 |
|:---|------|------|
| "商品超级周期 = 长期趋势机会" | 商品过去100年整体实际回报≈0，名义回报≈通胀 | 商品不做长期持有，只做周期择时+对冲配置 |
| "商品=通胀保护" | 仅在供给冲击型通胀中有效；需求拉动型通胀中股>商品 | 区分通胀类型：油价飙升+PCE↑=商品对冲有效；工资增长+PCE↑=商品对冲无效 |
| "商品=分散化工具" | 商品与股票在正常时期低相关，在流动性危机中高相关 | 正常期可分散，危机期不可依赖商品做股票对冲 |

### 1.2 为什么只分析原油？

| 商品 | 对冲价值 | 宏观信号 | 数据可得 | 与股票相关性 | 结论 |
|:---|:---:|:---:|:---:|:---:|------|
| **原油** | ✅ 供给冲击对冲 | ✅ 六维度验证 | ✅ 已有脚本 | 正常期低/危机期负 | **唯一纳入** |
| 黄金 | ✅ 货币信用对冲 | ✅ 三因子框架 | ✅ 已有框架 | 近零 | 已有独立框架 |
| 铜/铝 | 弱 | 中 | 可获取 | 正相关 | ❌ 主人排除 |
| 白银 | 弱 | 弱 | 可获取 | 正相关 | ❌ 主人排除 |
| 存储/光纤/新材料 | 无 | 弱 | 难获取 | 与科技股高度正相关 | ❌ 不具备对冲价值 |

## 二、原油分析

### 2.1 定位

> 原油是"宏观因子的物理标价"——油价不是被分析的对象，而是验证宏观判断的交叉指标。
> 原油+金油比+国债+黄金+股票 → 五类资产的交叉验证网络。

### 2.2 六维度分析

| 维度 | 指标 | 数据源 | 逻辑 |
|:---|------|:---|------|
| **宏观需求面** | 全球PMI（制造业）、中国原油进口量 | Bloomberg/海关 | PMI>50+进口↑=需求驱动油价↑ |
| **美元标价面** | DXY × 油价 滚动相关性 | FMP/Yahoo | 弱美元=油价标价支撑 |
| **供给冲击面** | OPEC+产量/配额比、美国页岩油钻机数 | EIA/Baker Hughes | 产量<配额+钻机↓=供给溢价 |
| **地缘风险面** | 霍尔木兹海峡通行量、中东冲突烈度 | 新闻/航运数据 | 海峡紧张→供给风险溢价↑ |
| **库存验证面** | EIA原油库存、OECD商业库存 | EIA/IEA | 库存↓+价格↑=真实供给紧张 |
| **期限结构面** | WTI/Brent远期曲线形态 | 期货数据 | Backwardation=现货紧张；Contango=供给充裕 |

### 2.3 金油比：宏观情绪的交叉温度计

> 黄金-原油比 = 避险需求 / 增长预期。金油比上升 = 市场定价"滞"大于"胀"，反之亦然。

| 金油比区间 | 宏观含义 | 对应情景 | 仓位含义 |
|:---|------|------|------|
| <15 | 强增长+弱避险 → 风险偏好极高 | 2005-2007, 2017-2019 | 超配股票，低配债券 |
| 15-20 | 正常均衡 | 大多数时期 | 均衡配置 |
| 20-25 | 温和避险 → 增长担忧上升 | 2015-2016, 2022H2 | 增配黄金，减仓股票 |
| 25-30 | 显著避险 → 衰退/地缘担忧 | 2008, 2020Q1 | 大幅减仓股票 |
| >30 | 极端避险 → 危机模式 | 罕见 | 现金+黄金为主 |

> ⚠️ **当前（2026-05-09）**：布伦特~$85，黄金现货~$3,650 → 金油比≈42.9。这个数值极高，但需注意：当前金价由央行购金+弱美元驱动而非恐慌，与2008/2020的"恐慌驱动"有本质区别。高频信号应拆分为金价≈货币政策对冲 + 油价≈增长预期两个独立维度。

### 2.4 与L1宏观评级的交叉验证规则

```
规则1: 油价↑ + L1通胀评分↑ → 验证宏观判断，维持L1评级
规则2: 油价↑ + L1通胀评分↓ → 矛盾信号，区分"供给冲击"vs"需求驱动的误判"
规则3: 油价↓ + L1金融条件↑ → 油价下跌改善金融条件，L1可能上调
规则4: 油价↑ + L1金融条件↓ → 滞胀风险，L1必须下调
规则5: 金油比↑ + VIX↓ → 矛盾信号（机构预判风险，散户未感知），警戒
```

### 2.5 油价情景与宏观含义

| 布伦特油价区间 | 宏观含义 | L1修正方向 | 对持仓影响 |
|:---|------|:---:|------|
| <$75 | 需求疲软确认 → 全球经济减速 | L1 CN↓ | 减仓新兴市场 |
| $75-90 | 正常区间 → 宏观中性 | 不变 | 维持 |
| $90-105 | 温和供给压力 → 通胀粘性 | L1 US↓（+0.1通胀权重） | 减仓高久期资产 |
| >$105 | 供给冲击 → 滞胀预警 | L1 US↓↓（+0.3通胀权重） | 大幅减仓股，加黄金 |

> ⚠️ **当前状态（2026-05-09）**：布伦特~$85（OPEC+增产+霍尔木兹海峡疏导），处于正常区间。PCE 3.2%不来自油价驱动，而来自工资+住房——这不是商品对冲有效的通胀类型。油价在给PCE降温。

## 三、五资产交叉验证网络

### 3.1 五类资产的对冲关系

```
              风险off（避险）         风险on（进攻）
             ←────────────────────────────────→

黄金 ─────────── 强对冲 ──────────── 机会成本承压
原油 ───── 供给冲击时对冲 ──── 增长驱动时同涨
美债 ──────── 核心避险 ────────── 利率上行承压
股票(QQQ) ──── 风险资产 ────────── 核心进攻
股票(CN)  ─ 新兴市场承压 ──────── CN独立行情时进攻
```

### 3.2 交叉验证规则矩阵

| 观察组合 | 信号 | 宏观含义 |
|:---|:---|------|
| 油价↑ + 金油比↑ | 矛盾——油价涨但避险更强 | 地缘溢价而非需求驱动 |
| 油价↓ + 金油比↓ | 增长预期改善 + 风险偏好回升 | 经济复苏信号 |
| 油价↑ + 10Y↑ + 黄金↑ | **滞胀预警**——三者同涨罕见 | 大幅减仓股票 |
| 油价↓ + 10Y↓ + 黄金↓ | 增长放缓 + 政策宽松预期 | 增配债券 |
| 油价↑ + DXY↓ | 美元定价的供给冲击 | 对非美投资者冲击放大 |

### 4.2 与三层多因子框架的整合

```
三层多因子框架 (L1+L2+L3) → 仓位中枢 + 风格偏好 + 中美配置
                              ↑
原油交叉验证 ← 油价方向 + 金油比 + 期限结构
                              ↑
                        修正L1评分
                       （±0.1-0.3调整）
```

### 4.3 交叉验证触发规则

```
IF 油价突破$105 → 强制重新评估L1通胀评分（无论是否矛盾）
IF 金油比突破25 + VIX<20 → 矛盾信号，预判风险事件
IF 油价↑ + 10Y↑ + 黄金↑ 三者同涨 → 滞胀模式，L1 US必须下调≥0.5
IF 油价↓ + 10Y↓ + 黄金↓ 三者同跌 → 通缩/衰退模式，加仓债券
```
