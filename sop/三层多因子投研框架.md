# 三层多因子投研框架 v1.0

> 设计原则：宏观定仓位 → 风格定方向 → 情绪定节奏
> 每层独立评分(1-5)，最终加权给出仓位建议

---

## 架构总览

```
┌──────────────────────────────────────────────────┐
│  L1 宏观环境 (中美双轨)  →  仓位中枢             │
│  US: 就业/通胀/货币/金融条件/地缘 (5因子)        │
│  CN: 制造业/消费/信贷/房地产/政策 (5因子)        │
├──────────────────────────────────────────────────┤
│  L2 市场风格 (3因子)  →  配置倾斜                │
│  动量 · 价值 · 成长                               │
├──────────────────────────────────────────────────┤
│  L3 情绪温度 (3因子)  →  进出节奏                │
│  市场温度 · 风险偏好 · 资金参与                   │
└──────────────────────────────────────────────────┘
         ↓
  综合仓位建议 + 风格偏好 + 中美配置比例
```

### L2 风格定位

| 维度 | 得分 | 含义 |
|---|---|---|
| 动量 | **5.00** | 趋势极强，顺势而为 |
| 价值 | **1.50** | 极度高估，安全边际为零 |

> **风格判断：强动量+强成长+弱价值 = 典型的"成长股牛市后期"特征。**
> 这个组合历史上出现在1998-1999年（互联网泡沫）和2020-2021年（零利率泡沫）。
> 配置偏好：成长>动量>>价值

## 综合多因子评分卡

```
┌────────────┬───────┬──────────────────────────────────┐
│ 层次       │ 得分  │ 解读                             │
├────────────┼───────┼──────────────────────────────────┤
│ L1 US宏观  │ 2.90  │ 中性偏谨慎                      │
│ L1 CN宏观  │ 3.02  │ 中性偏弱复苏                    │
│ L1 综合    │ 2.95  │ 中美加权(60/40)：中性偏谨慎      │
│ L2 动量    │ 5.00  │ 趋势极强，顺势做多              │
│ L2 价值    │ 1.50  │ 极度高估，无安全边际            │
│ L2 成长    │ 4.25  │ 增长强劲，支撑高估值            │
│ L3 温度    │ 3.75  │ 偏热，但未到极端                │
│ L3 偏好    │ 3.00  │ 散户未狂热，泡沫有空间          │
│ L3 资金    │ 3.50  │ 资金活跃，未过热                │
├────────────┼───────┼──────────────────────────────────┤
│ 综合加权   │ 3.28  │ 中性偏积极                      │
└────────────┴───────┴──────────────────────────────────┘

加权公式：L1×40% + L2均值×30% + L3均值×30%
L1 = US(2.90)×0.6 + CN(3.02)×0.4 = 1.74 + 1.21 = 2.95
综合 = 2.95×0.4 + 3.58×0.3 + 3.42×0.3
     = 1.18 + 1.07 + 1.03
     = 3.28
```

## 当前综合判断（2026-05-08）

**仓位中枢：55-60%**（中等偏高，留40%+现金应对尾部风险）

**核心矛盾：**
- ✅ 动量极强 + 成长强劲 + 散户未狂热 = 趋势延续的动力
- ⚠️ 估值极端（CAPE 36.48 仅次1999年）+ VIX可能随时跳升 = 下行风险
- 🔑 **这不是撤退的信号，但也不是加仓的信号。这是"持有并收紧止损"的信号。**

**情景推演：**

| 情景 | 概率 | L1 变化 | L2 变化 | L3 变化 | 仓位调整 |
|---|---|---|---|---|---|
| CPI温和+AI盈利超预期 | 30% | US +0.3 | 动量保持 | 温度↑ | +10%至65-70% |
| CPI高+Fed转鹰 | 20% | US -0.5 | 动量↓ | 温度↑ | -15%至40-45% |
| 维持现状(软着陆) | 30% | 持平 | 动量保持 | 偏好↑ | 维持55-60% |
| 中国消费改善+PBOC宽松 | 10% | CN +0.8 | 成长↑ | 偏好↑ | 中国仓位+10% |
| 黑天鹅(中东升级) | 10% | US -1.0, CN -0.5 | 全↓ | 恐慌→机会 | -25%至30-35% |
